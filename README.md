# Ray Caster

Hardest part is refraction. My refraction code works but numbers not fit.

## Requirements

- CMake
- Build Tool (Ninja preffered)
- Compiler (MSVC preffered)
- vcpkg

## How to compile?

```sh
cmake . out -G Ninja
cd Build/[cmake preset name]
ninja
```

The binary file located in Bin/RayCaster
